# SecurePass

MVP frontend de una app generadora de contrasenas creada con React, Vite,
Tailwind CSS, componentes shadcn/ui, lucide-react y sonner.

## Instalacion

```bash
npm install
npm run dev
```

Abre la URL local que muestre Vite, normalmente:

```bash
http://localhost:5173/
```

## Build

```bash
npm run build
```

## Estructura principal

```text
src/
  App.jsx
  components/
    PasswordDisplay.jsx
    PasswordOptions.jsx
    StrengthMeter.jsx
    ui/
  utils/
    passwordGenerator.js
  lib/
    utils.js
```
